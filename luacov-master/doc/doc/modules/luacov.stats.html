Module luacov.stats

Manages the file with statistics (being) collected.

Functions

load (statsfile) Loads the stats file.

Functions

load (statsfile)
Loads the stats file.

Parameters:

  • statsfile path to the stats file.

Returns:

    table with data or nil if couldn't load. The table maps filenames to stats tables. Per-file tables map line numbers to hits or nils when there are no hits. Additionally, .max field contains maximum line number and .max_hits contains maximum number of hits in the file.
save (statsfile, data)
Saves data to the stats file.

Parameters:

  • statsfile path to the stats file.
  • data data to store.
